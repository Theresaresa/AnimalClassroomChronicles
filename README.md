# Animal Classroom Chronicles: The Golden Speaking Ball Mystery

## Project Description

Animal Classroom Chronicles is a 2D educational detective visual novel developed in Unity.

Players investigate the theft of the Golden Speaking Ball while solving English language learning tasks.

## Features

- Avatar selection
- Multiple chapters
- Reading tasks
- Listening tasks
- Grammar exercises
- Vocabulary exercises
- Progress saving using PlayerPrefs
- Chapter unlock system
- Detective certificate

## Technologies

- Unity 6
- C#
- TextMeshPro
- GitHub
